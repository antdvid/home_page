This function insert state to a Dirichlet boundary. The six (four for 2D) sides of the boundary memory has been allocated, this function fills in the content of the boundary state. It provide two methods to set the boundary: (a) set the boundary with a constant state, in this case, the address of the state as a pointer must be supplied as a pointer while state_func, state_func_name and state_func_params must be set to NULL; (b) set the boundary state through a state function, in this case, state must be set to NULL while state_func, state_func_name and state_func_params must be supplied. Currently only flow-through and time-dependent functions are available.

Parameters
frontinout Pointer to Front.
state_funcin Function pointer where state is evaluated in option (b).
state_func_namein Function name as a char string, for option (b).
state_func_paramsin Associated function parameters, for option (b).
statein The address of the constant state, for option (a).
hsinout The pointer to the boundary as a hyper surface structure.
indexThe boundary surface index.
void FT_MixedBoundaryHypSurfs ( INTERFACE *  intfc,

int  nb,

This function returns a set of hyper surfaces (curves in 2D and surfaces in 3D) whose wave type matches the input w_type. If the boundary on idir and side nb is not MIXED_TYPE_BOUNDARY, the function returns NULL. Otherwise it will return an array of hyper surfaces. The function also assign num_hs, the total number of hyper surfaces in the returned set.

Parameters
intfcin POINTER to the input interface.
idirin Direction 0 for x-dir, 1 for y-dir, 2 for z-dir.
nbin Side, 0 for lower side, 1 for upper side.
w_typein Wave type of the hyper surfaces looking for.
num_hsout Number of hyper surfaces in the matching set.
